# Supertonic3 TTS Add-on

Ultra-fast, on-device text-to-speech for Home Assistant powered by Supertonic-3.

## About

This add-on brings Supertonic-3 TTS to Home Assistant, providing:

- **Lightning-fast** on-device synthesis (several times real-time on consumer CPUs)
- **Complete privacy** — 100% on-device processing, no cloud, no API calls
- **Multilingual support** - 31 languages supported natively
- **10 voice options** — 5 male and 5 female voices
- **Streaming** — Playback starts on the first sentence, no waiting for the full text
- **Smart abbreviations** — FR/EN abbreviations expanded to spoken form before synthesis

Perfect for home automation announcements, notifications, and accessibility features.

## Installation

1. Add this repository to your Home Assistant add-on store
2. Install the "Supertonic3 TTS" add-on
3. Configure your preferences (language, voice, speed, etc.)
4. Start the add-on (first start downloads/loads ~385 MB of models)
5. **That's it!** The TTS service is automatically discovered via Wyoming protocol
